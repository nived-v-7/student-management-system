Core Objective

Project Overview

Nexus Student Management System replaces outdated administrative sheets with a lightning-fast, secure, and beautiful centralized full-stack dashboard.

  • Simplified Administration: Single point of control for course registrations, roster modifications, and grade sheets.
  • Analytical Insights: Instant distribution calculations and performance average tracking.
  • Dynamic Design: Clean obsidian glassmorphism panels offering responsive, desktop-premium interfaces.

Tech Stack

System Architecture

Our stack prioritizes lightweight dependencies, rapid load times, and easy local execution on any system configuration.

  • Express & Node.js Server: REST APIs serving structured endpoints with CORS configurations.
  • JSON Data persistence: File-based persistence utilizing a self-repairing local JSON database structure.
  • Vanilla CSS Design: Avoids heavy frameworks, utilizing CSS variables, responsive grids, and backdrop blur.

Module 1

Overview Dashboard

The central hub aggregates records into executive KPIs and logs background actions in real-time.

  • Executive Indicators: Tally panels tracking total students, active courses, and enrollments.
  • Enrollment Graphs: Beautiful SVG-drawn bar charts displaying department distributions dynamically.
  • Action Feeds: Event logger mapping recent actions (e.g. grade updates, additions) with "time ago" formatting.

Module 2

Student Directory

A high-performance student database layout supporting extensive filter and query combinations.

  • Real-time Search: Instant filters that search matching student names, emails, or IDs.
  • Enrollment Badges: Status tags mapping active, suspended, or graduated students in color-coded pills.
  • Pop-up Modals: Standard, unified modals supporting profile creations and editing actions.
Module 3

Course Catalog

Curriculum records are presented in modern visual cards listing credit tallies and assignees.

  • Instructor Mapping: Quick visual references of director-instructors (e.g. Dr. Alan Turing).
  • Student counters: Displaying the number of active enrollments in each catalog block.
  • Detail Side-Drawer: A slide-in drawer layout displaying the roster of enrolled students without page reload.

Module 4

Grade Portal

Enables grading administrators to evaluate student outcomes and update sheets inline.

  • Inline Autosave: Direct grade number cell inputs. Triggers instant database updates on click-out (blur) or Enter key.
  • GPA Calculations: Automated calculations for letter symbols (A, B, C, D, F) and 10.0 scale GPA scores.
  • Performance spreads: Breakdown panels outlining aggregate grade letters.

Backend Interfaces

Database Schema & REST APIs

Our REST interfaces are organized logically to ensure complete security, scalability, and easy client mappings.

/api/students

GET: Fetch student directory with GPA summaries.
POST: Register profile.
DELETE: Remove student.

/api/courses

GET: Fetch course grid with tally counters.
POST: Create course.
DELETE: Remove course.

/api/grades

GET: Flattened list of course grades.
PUT: Insert or update numeric scores inline.

Aesthetics

UI & UX Design Philosophy

Nexus Academy features premium visual parameters tailored to wow administrators at first glance.

  • Obsidian Glassmorphism: Backdrop-blur panels blending borders and dark radial backgrounds.
  • Fluid Animation: CSS transit coordinates ensuring spring-loaded triggers for clicks, card hovers, and modals.
  • Contrast & Focus: Deep dark backgrounds with high-contrast text and glowing badges.

Roadmap

Future Enhancements
