Nexus Student Management System - Desktop Copy
=============================================

This folder contains the self-contained, lightweight source code files for your project.

It runs on standard Node.js with ZERO external dependencies (no Express or CORS modules needed, and no node_modules folder is required).

📁 Project Structure:
- public/          -> Frontend files (HTML, CSS, JS, illustrations)
- server.js        -> Zero-dependency backend server
- database.json    -> Local database storage
- README.txt       -> This guide

🚀 How to Run this project from your Desktop:
1. Open VS Code.
2. Click "File" > "Open Folder..." and select this Desktop folder:
   C:\Users\Lenovo\Desktop\student-management-system
3. Open a terminal in VS Code (Ctrl + ` or Terminal > New Terminal).
4. Run the server directly:
   node server.js
5. Open your browser and navigate to:
   http://localhost:3000
